# Turl Street Orchestra (TSO) Website

## Overview
This repository contains the front-end website files and promotional assets for the **Turl Street Orchestra**, Oxford's premier non-auditioning student orchestra. The site serves as a central hub for prospective members, current musicians, and concert attendees to find information about rehearsals, concerts, and how to get involved.

## Project Files

* **`index.html`**: The primary landing page for the orchestra. This version features an animated starfield background, a spinning gold ring animation around the logo, and a "Stay Connected" section with links to the mailing list and social media.
* **`tsowebsite.html`**: An alternative, earlier version of the main website. It shares the same core styling and structure as `index.html` but does not include the logo integration in the header or the distinct social media connection links.
* **`462661142_576296568172238_7560912236331560459_n.jpg`**: The official Turl Street Orchestra logo, stylized with a classic black-and-white motif of a string instrument and the letter 'T'. It is used as the favicon and main header image in `index.html`.
* **`background.png`**: An illustrated poster detailing a specific repertoire, prominently featuring Rossini's "Barber of Seville Overture" and Beethoven's "Pastorale".

## About the Orchestra
The Turl Street Orchestra websites provides the following details:
* **Rehearsals**: Fridays from 7:30 PM to 10:00 PM at Exeter College Chapel during term time (Weeks 1 to 7).
* **Concerts**: Three free concerts are held annually (one per term) at the Exeter College Chapel. 
* **Membership**: The orchestra is non-auditioning and open to students of all musical abilities. 

## Setup & Usage
This is a static HTML/CSS website. No backend server or build process is required. 
1. Clone or download the repository files.
2. Ensure `462661142_576296568172238_7560912236331560459_n.jpg` is located in the same directory as `index.html` so the logo and favicon load correctly.
3. Open `index.html` in any modern web browser to view the site.
